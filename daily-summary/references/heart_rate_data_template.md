# 心率数据存储模板

本文件说明如何存储心率数据以便 daily-summary skill 正确分析。

## 目录结构

```
Obsidian Vault/
├── Areas/
│   ├── Running/                    # Garmin 跑步数据（自动生成）
│   │   ├── 节奏跑/
│   │   ├── 间歇跑/
│   │   └── ...
│   └── Health/                     # 健康数据（需手动或快捷指令创建）
│       ├── WHOOP/                  # WHOOP 数据
│       │   └── YYYY-MM-DD.md
│       ├── AppleWatch/             # Apple Watch 数据
│       │   └── YYYY-MM-DD.md
│       └── 心率警报历史.md          # 警报记录（自动生成）
```

## WHOOP 数据格式

文件路径：`Areas/Health/WHOOP/2026-03-25.md`

```markdown
# WHOOP 数据 - 2026-03-25

## 恢复指标
- HRV: 72 ms
- RHR: 56 bpm
- 恢复分数: 78%

## 睡眠数据
- 睡眠时长: 7h 23m
- 呼吸频率: 14.2
- SpO2: 98%

## 备注
- 昨晚睡眠质量良好
```

## Apple Watch 数据格式

文件路径：`Areas/Health/AppleWatch/2026-03-25.md`

```markdown
# Apple Watch 健康数据 - 2026-03-25

## 心率数据
- 静息心率: 52 bpm
- 步行平均心率: 95 bpm

## ECG 记录
- 时间: 08:30

- 备注: 无异常

## 通知
- 不规则心律通知: 无

## 备注
- 今日感觉良好
```

## 异常记录示例

### 不规则心律通知

```markdown
# Apple Watch 健康数据 - 2026-03-25

## ⚠️ 警报
- **不规则心律通知**: 是
- **时间**: 14:23
- **活动状态**: 静坐

## 建议行动
- [ ] 前往医院急诊检查
- [ ] 预约心内科门诊
```

### ECG 检测到房颤

```markdown
# Apple Watch 健康数据 - 2026-03-25

## ⚠️ ECG 结果
- **结果**: 房颤(AFib)
- **时间**: 09:15
- **截图**: ![ECG截图](attachments/ecg_20260325.png)
```

```markdown

## ECG 记录

- 建议就医检查
```

## 快捷指令自动化建议

可以使用 iOS 快捷指令自动创建每日健康数据笔记：

### 1. WHOOP 快捷指令
**触发**: 每日早晨 7:00
**动作**:
1. 读取 WHOOP 应用数据
2. 创建 Markdown 文件到 `Areas/Health/WHOOP/YYYY-MM-DD.md`

### 2. Apple Watch 快捷指令
**触发**: 每日晚上 22:00
**动作**:
1. 读取健康应用数据（静息心率、HRV、血氧）
2. 创建 Markdown 文件到 `Areas/Health/AppleWatch/YYYY-MM-DD.md`

### 3. 异常警报快捷指令
**触发**: 收到不规则心律通知时
**动作**:
1. 立即创建警报笔记
2. 发送 Telegram 通知给 DaoBot
3. 添加紧急待办到今日笔记

## 数据字段说明

| 字段 | 来源 | 格式 | 说明 |
|------|------|------|------|
| HRV | WHOOP | `HRV: 72 ms` | 心率变异性，单位毫秒 |
| RHR | WHOOP | `RHR: 56 bpm` | 静息心率，单位 bpm |
| SpO2 | WHOOP | `SpO2: 98%` | 血氧饱和度，单位 % |
| 呼吸频率 | WHOOP | `呼吸频率: 14.2` | 单位 次/分 |
| 静息心率 | Apple Watch | `静息心率: 52 bpm` | 单位 bpm |
| ECG结果 | Apple Watch | `结果: 窦性心律` | 可选：窦性心律/房颤/窦性心律不齐 |
| 不规则心律通知 | Apple Watch | `不规则心律通知: 是` | 触发紧急警报的关键字段 |

## 警报触发条件

### 🔴 紧急信号
1. **Apple Watch 不规则心律通知**: 内容为 `不规则心律通知: 是`
2. **Apple Watch ECG 房颤**: 内容包含 `房颤` 或 `AFib`
3. **运动中极高心率**: 最大心率 > 200 bpm
4. **运动中极低心率**: 最大心率 < 50 bpm

### 🟡 早期预警
1. **HRV 连续3天暴跌 > 30%**
2. **RHR 连续3天升高 > 10 bpm**
3. **静息心率连续3天 > 100 或 < 40**
4. **血氧 < 90%**
5. **运动中高心率**: 190-200 bpm

### 🟢 长期趋势
1. **HRV 3个月趋势下行**
2. **呼吸频率持续 > 18 次/分**
3. **窦性心律不齐频繁出现**（3次以上）
