# iOS 快捷指令配置指南

本文件提供 iOS 快捷指令的配置示例，用于自动化心率数据记录。

## 快捷指令 1: WHOOP 每日数据记录

### 配置步骤

1. **打开快捷指令 App** → 创建个人自动化
2. **触发条件**: 特定时间（每天 07:00）
3. **添加操作**:

```
操作 1: 获取剪贴板
  （手动从 WHOOP App 复制数据后运行）

操作 2: 文本
  内容:
  # WHOOP 数据 - {{当前日期}}

  ## 恢复指标
  - HRV: [粘贴HRV值] ms
  - RHR: [粘贴RHR值] bpm
  - 恢复分数: [粘贴恢复分数]%

  ## 睡眠数据
  - 睡眠时长: [粘贴睡眠时长]
  - 呼吸频率: [粘贴呼吸频率]
  - SpO2: [粘贴血氧值]%

操作 3: 创建备忘录
  文件夹: Obsidian/WHOOP
  文件名: {{当前日期}}.md
  内容: 操作2的文本
```

## 快捷指令 2: Apple Watch 健康数据记录

### 配置步骤

1. **打开快捷指令 App** → 创建个人自动化
2. **触发条件**: 特定时间（每天 22:00）
3. **添加操作**:

```
操作 1: 获取健康样本
  类型: 静息心率
  排序: 最新
  限制: 1

操作 2: 获取健康样本
  类型: 心率变异性 (HRV)
  排序: 最新
  限制: 1

操作 3: 获取健康样本
  类型: 血氧
  排序: 最新
  限制: 1

操作 4: 文本
  内容:
  # Apple Watch 健康数据 - {{当前日期}}

  ## 心率数据
  - 静息心率: [操作1结果] bpm

  ## HRV 数据
  - 最新 HRV: [操作2结果] ms

  ## 血氧数据
  - SpO2: [操作3结果]%

  ## ECG 记录
  - 结果: 窦性心律
  - 时间: {{当前时间}}

  ## 通知
  - 不规则心律通知: 无

操作 5: 创建备忘录
  文件夹: Obsidian/AppleWatch
  文件名: {{当前日期}}.md
  内容: 操作4的文本
```

## 快捷指令 3: 不规则心律警报

### 配置步骤

1. **打开快捷指令 App** → 创建个人自动化
2. **触发条件**: 收到通知（包含"不规则心律"）
3. **添加操作**:

```
操作 1: 获取当前日期和时间

操作 2: 文本
  内容:
  # ⚠️ Apple Watch 健康数据 - {{当前日期}}

  ## 🔴 紧急警报
  - **不规则心律通知**: 是
  - **时间**: {{当前时间}}
  - **活动状态**: [手动填写]

  ## 建议行动
  - [ ] 立即前往医院急诊
  - [ ] 联系家人/朋友
  - [ ] 保存 ECG 截图

操作 3: 创建备忘录
  文件夹: Obsidian/AppleWatch
  文件名: {{当前日期}}-警报.md
  内容: 操作2的文本

操作 4: 发送消息
  应用: Telegram
  收件人: DaoBot
  内容: 🚨 紧急！Apple Watch 检测到不规则心律，请立即就医！

操作 5: 播放声音
  声音: 警报声
  重复: 3次
```

## 快捷指令 4: ECG 记录快捷方式

### 配置步骤

1. **创建快捷指令**（手动运行）
2. **添加操作**:

```
操作 1: 从菜单选取
  选项:
  - 窦性心律（正常）
  - 窦性心律不齐
  - 房颤(AFib)
  - 不确定

操作 2: 要求输入
  提示: 是否有心悸、胸闷等症状？

操作 3: 获取当前日期和时间

操作 4: 如果 [操作1结果] 是 "房颤(AFib)"
  则:
    - 文本: [创建紧急警报内容]
    - 发送消息给 DaoBot
  否则:
    - 文本: [创建普通记录内容]

操作 5: 创建备忘录
  文件夹: Obsidian/AppleWatch
  文件名: {{当前日期}}-ECG.md
```

## 自动化建议

### 推荐设置

| 快捷指令 | 触发方式 | 频率 |
|---------|---------|------|
| WHOOP 数据记录 | 定时 | 每日 07:00 |
| Apple Watch 数据 | 定时 | 每日 22:00 |
| 不规则心律警报 | 通知触发 | 即时 |
| ECG 记录 | 手动 | 每次测量后 |

### 注意事项

1. **健康数据权限**: 确保快捷指令有访问健康数据的权限
2. **Obsidian 同步**: 使用 iCloud 同步确保数据及时同步到 Mac
3. **通知权限**: 允许快捷指令发送通知和消息
4. **手动补充**: 快捷指令只能获取部分数据，关键信息需要手动补充

## 替代方案：使用 Siri 语音记录

如果不想配置复杂快捷指令，可以使用 Siri 语音快速记录：

```
"嘿 Siri，记录心率"
→ 打开 Obsidian 快速笔记
→ 语音输入心率数据
→ 保存到指定文件夹
```
