# 招商银行信用卡账单格式说明

## PDF 结构

招商银行信用卡账单 PDF 通常包含以下部分：

### 1. 账单头部

```
招商银行信用卡对账单（个人消费卡账户 2025年02月）
CMB Credit Card Statement (2025.02)
```

### 2. 账户信息

- 账单日 (Statement Date)
- 到期还款日 (Payment Due Date)
- 信用额度 (Credit Limit)
- 本期应还金额 (New Balance)
- 本期最低还款额 (Min. Payment)

### 3. 交易明细 (Transaction Details)

#### 人民币账户 (RMB A/C)

| 字段 | 说明 |
|------|------|
| 交易日 (Trans Date) | 交易发生日期 (MM/DD) |
| 记账日 (Post Date) | 入账日期 (MM/DD) |
| 交易摘要 (Description) | 交易描述 |
| 人民币金额 (RMB Amount) | 交易金额（正数为消费，负数为还款） |
| 卡号末四位 (Card Number) | 信用卡后四位 |
| 交易地金额 (Original Trans Amount) | 原始交易金额 |

#### 外币账户 (Foreign Currency A/C)

如果有外币交易，会单独列出。

### 4. 分期付款明细

如果有分期，会显示：
```
消费分期-商户名称 本金 第X/Y期
```

### 5. 费用说明

- 年费
- 利息
- 手续费
- 其他费用

## 正则表达式模式

### 提取账单信息

```python
patterns = {
    "statement_date": r"账单日\s*(\d{4}年\d{2}月\d{2}日)",
    "due_date": r"到期还款日\s*(\d{4}年\d{2}月\d{2}日)",
    "credit_limit": r"信用额度\s*¥\s*([\d,]+\.\d{2})",
    "new_balance": r"本期应还金额\s*¥\s*([\d,]+\.\d{2})",
    "min_payment": r"本期最低还款额\s*¥\s*([\d,]+\.\d{2})",
}
```

### 提取交易记录

```python
# 格式: 交易日 [记账日] 描述 金额 卡号
trans_pattern = r"(\d{2}/\d{2})\s+(\d{2}/\d{2})?\s+(.+?)\s+([-\d,]+\.\d{2})\s+(\d{4})"
```

## 特殊情况处理

### 1. 还款记录

- 金额为负数
- 描述通常包含"还款"、"转账"等关键词

### 2. 分期付款

- 描述格式: `消费分期-商户名称 本金 第X/Y期`
- 需要单独识别和统计

### 3. 年费

- 通常在账单日当天记账
- 可能有积分抵扣（显示为负数）

### 4. 退款

- 金额为负数
- 描述可能与原消费相同

### 5. 外币交易

- 会显示原始币种和金额
- 人民币金额为汇率转换后的值

|------|--------|
| 餐饮 | 餐、饭、食、咖啡、茶、酒、美团、饿了么、肯德基、麦当劳、星巴克 |

## 注意事项

1. **日期格式**: 交易日期使用 MM/DD 格式，需要结合账单月份推断完整日期
2. **金额格式**: 可能包含千位分隔符（逗号），需要去除后转换为浮点数
3. **编码问题**: PDF 中的中文可能有编码问题，需要正确处理 UTF-8
4. **多页账单**: 交易记录可能跨多页，需要完整提取
5. **补充账单**: 文件名可能包含"（补）"标记，表示补发的账单
