# 每日信用管家账单格式说明

## 邮件格式

招商银行"每日信用管家"邮件包含以下信息：

### 邮件主题
```
每日信用管家
```

### 邮件内容结构

```
截至昨日最后一笔交易，您的额度和积分信息如下：
￥XX,XXX.XX    可用额度
XX,XXX         积分余额

2026/03/14 您的消费明细如下：

时间      金额       卡号     类型   商户
00:42:16  CNY 3.00   尾号 2693  消费   支付宝 - 北京鸿易博科技有限公司
01:00:11  CNY 14.83  尾号 2693  消费   支付宝 - 北京嘀嘀无限科技发展有限公司
16:06:36  USD 20.88  尾号 5962  邮购   VERCEL +19513836898 CA US
```

## 交易字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 时间 | 交易发生时间（HH:MM:SS） | 00:42:16 |
| 币种 | 交易币种 | CNY, USD, HKD |
| 金额 | 交易金额（原币） | 3.00, 20.88 |
| 卡号 | 信用卡尾号 | 2693, 5962 |
| 类型 | 交易类型 | 消费，邮购，退款，退货 |
| 商户 | 商户名称或支付渠道 | 支付宝 - 北京鸿易博科技有限公司 |

## 支持的交易类型

- **消费**: 普通信用卡消费
- **邮购**: 海外邮购交易
- **退款**: 商户发起的退款
- **退货**: 退货交易

## 外币汇率

当前支持的汇率配置（在 `cmb_common.py` 中）：

| 币种 | 汇率 | 说明 |
|------|------|------|
| CNY | 1.0 | 人民币（基准） |
| USD | 7.25 | 美元兑人民币 |
| HKD | 0.93 | 港币兑人民币 |
| EUR | 7.85 | 欧元兑人民币 |
| JPY | 0.048 | 日元兑人民币 |
| GBP | 9.15 | 英镑兑人民币 |

## 解析规则

### 正则表达式
```python
pattern = r'(\d{2}:\d{2}:\d{2})\s*(CNY|USD|HKD)\s*([\d,]+\.\d{2})\s*尾号\s*(\d+)\s*(消费 | 邮购 | 退款 | 退货)\s*(.+?)(?=\d{2}:\d{2}:\d{2}|$)'
```

### 解析流程
1. 提取 HTML 中的文字内容
2. 替换 `&nbsp;` 为空格
3. 使用正则表达式匹配交易记录
4. 将外币金额转换为 CNY
5. 按卡号和分类汇总统计

## 输出文件格式

**文件名**: `YYYYMMDD-信用卡日账单.md`

**内容结构**:
- 账单概况（日期、统计周期）
- 交易汇总（按卡号分组）
- 消费分类统计
- 交易明细（按卡号分组）
- 消费建议

## 使用示例

```bash
# 拉取最近 1 天
python3 scripts/cmb-daily-statement.py

# 拉取最近 7 天汇总
python3 scripts/cmb-daily-statement.py --days 7

# 指定日期
python3 scripts/cmb-daily-statement.py --date 2026-03-14

# 自定义输出目录
python3 scripts/cmb-daily-statement.py --output-dir /path/to/output
```

## 与月度账单的区别

| 特性 | 每日交易 | 月度账单 |
|------|---------|---------|
| 邮件主题 | 每日信用管家 | 信用卡电子账单 |
| 数据粒度 | 单日明细 | 整月汇总 |
| 包含额度信息 | 是 | 是 |
| 包含分期信息 | 否 | 是 |
| 外币转换 | 自动转 CNY | 原币显示 |
| 输出文件 | YYYYMMDD-信用卡日账单.md | YYYYMM-信用卡账单分析.md |
